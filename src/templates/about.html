About Moderator

What is this for?

Moderator is a tool to make it easy to ask pointed questions. It was built to replace the now defunct Google Moderator. Anyone can post questions completely anonymously. Questions can be voted on, questions with higher scores appear closer to the top of the page. At Appboy, weâ€™ll use this tool to drive Q&A during all-hands.

How anonymous am I?

Questions are posted anonymously: every event has a unique link that can be used to post questions, we save the questions and throw out any other incidental data that may have been associated with it.

Comments are not anonymous and will have your name on them. Our system keeps track of votes (to prevent double voting and allow you to change your vote), but will not ever show the identity of who voted for a question or comment anywhere.

This was a side project. The source code is available publicly if you or a technically minded friend wants to verify any of these claims.
